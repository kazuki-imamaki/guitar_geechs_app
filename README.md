## 制作背景

学生時代からの趣味であるギターを題材にしたサービスを制作しました。

好きなギタリストの画像や映像などを投稿し共有することができます。

## 使用技術
* Ruby 2.6.3
* Ruby on Rails 6.0.3
* PostgreSQL
* JavaScript (jQuery)
* Bootstrap
* AWS (cloud9,VPC,EC2,S3)
* Git, GitHub
* Heroku
* Mailgun

## 機能一覧
* ユーザー認証機能
* ログイン機能
* ゲストユーザーログイン機能 
* 投稿機能
* 画像アップロード機能
* 動画アップロード機能
* いいね機能
* フォロー機能
* ページネーション機能
